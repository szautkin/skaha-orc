# UVic Color Palette

| Token | Hex | Usage |
|-------|-----|-------|
| Congress Blue | `#005493` | Primary buttons, links, active states |
| Prussian Blue | `#002754` | Header bar, sidebar, dark backgrounds |
| Buttercup Yellow | `#f5aa1c` | Accent, warnings, deploying status |
| Tall Poppy Red | `#c63527` | Errors, destructive actions, failed status |
| Success Green | `#16a34a` | Healthy/deployed status |
| Light Blue | `#e8f4fd` | Card backgrounds, hover states |
| Neutral Gray | `#64748b` | Muted text, borders |
